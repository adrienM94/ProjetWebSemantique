# ProjetWebSemantique
 * **Année** : 2021
 * **Sujet** : Football

## Modélisation
 Nous mettons en place un modèle lié au sujet football, en particulier nous mettons en place la description d'un match.
 
 * Mise en place d'un [**diagramme de classes UML**](https://github.com/adrienM94/ProjetWebSemantique/blob/main/Diagramme_classes_UML_projet.png) afin de montrer les liaisons entre les ressources et les caractéristique clés de ces ressources.
 * Mise en place d'un [**diagramme d'objets UML**](https://github.com/adrienM94/ProjetWebSemantique/blob/main/Diagramme_objet_UML_projet.png) pour donner une instance du diagramme de classes.
 * Mise en place du [**graphe RDFS**](https://github.com/adrienM94/ProjetWebSemantique/blob/main/Graphe_RDFS_projet.png) pour la description des ressources et des liaisons entre elles dans le cadre du web sémantique.  
 * Mise en place du [**graphe OWL**](https://github.com/adrienM94/ProjetWebSemantique/blob/main/Graphe_OWL_projet.png) pour définir une ontologie propre au modèle que nous avons créé.
